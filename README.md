DUPILCATE VALUES AND MISTAKES WERE CORRECTED USING:
```` SQL
ALTER TABLE ORDERS_INFO
DROP COLUMN PRICE_PER_UNIT

UPDATE PRODUCTS
SET PRODUCT_ID = 101
WHERE PRODUCT_ID = 1001
````
